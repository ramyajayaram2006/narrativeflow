\*\*NarrativeFlow\*\* is an interactive story co-writing application that combines human creativity with AI assistance. Built with Python and Streamlit, it allows you to write stories collaboratively with local LLMs (Llama 3.2 via Ollama) or cloud AI providers (OpenAI, Claude).



\## ✨ Features



\### Core Capabilities

\- \*\*🤖 AI-Powered Co-Writing\*\* - Continue sentences, add paragraphs, or let Smart Choice detect your intent

\- \*\*📚 Story Management\*\* - Create, organize, and delete stories with genre and tone settings

\- \*\*🔒 Secure Authentication\*\* - bcrypt password hashing with automatic SHA-256 migration

\- \*\*💾 Persistent Storage\*\* - SQLite database saves users, stories, characters, and scenes

\- \*\*📄 Word Export\*\* - Generate formatted Word documents from your stories

\- \*\*🏠 Local-First\*\* - Run completely offline with Ollama (no internet required after setup)



\### Milestone 3 Enhancements

\- \*\*🎭 Character Consistency\*\* - Create character profiles with speaking styles that persist across sessions

\- \*\*🏞️ Scene Management\*\* - Organize stories by scenes with location and purpose tracking

\- \*\*📊 Writing Analytics\*\* - Track average sentence length, reading level, and word frequency

\- \*\*🔍 Live Search\*\* - Search across story titles and content

\- \*\*⚡ Live Token Streaming\*\* - Watch AI responses appear word-by-word

\- \*\*🚫 Story-Only Mode\*\* - Classifier that blocks off-topic inputs (math, coding, general knowledge)
